Mind-Bending Puzzles

The Ship of Theseus

If you replace every plank of a ship, is it the same ship? What if you build a new ship from the old planks? Which one is the original?

The Simulation Argument

If advanced civilisations can run simulations of reality, statistically we're more likely to be IN a simulation than not. Are you real?

The Bootstrap Paradox

You travel back in time and give Shakespeare his own plays. He copies them. Where did the plays come from? Neither of you wrote them.

The Last Question
